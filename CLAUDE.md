# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Repo Is

A curated collection of **free** online robotics educational resources (tutorials, courses, textbooks, documentation) maintained by [Robotics Career Coach](https://roboticscareercoach.com). This is a content-only repository — no build system, tests, or application code. All work involves editing Markdown files.

## Repository Structure

- **[README.md](README.md)** — Landing page with topic index and contribution guidelines
- **[learning-path.md](learning-path.md)** — Staged week-by-week learning plan (Foundations → Core Robotics → Specializations → Capstone)
- **[topics/](topics/)** — One Markdown file per topic area (ROS 2, software engineering, computer vision, embedded systems, mechanical engineering & CAD, ML & AI, cloud & DevOps, simulation)

## Content Conventions

- Every listed resource must be **free to access** (audit-for-free Coursera/edX courses qualify; note the audit option when it applies).
- Each resource entry includes: **title**, **author/institution**, **URL**, and a brief description of what it covers and its difficulty level.
- Credit the original authors, instructors, and institutions that created each resource.
- When adding a new topic, create a file in `topics/`, add a row to the table in `README.md`, and integrate it into the learning path in `learning-path.md` if appropriate.
