# Embedded Systems

Resources for microcontrollers, firmware, real-time systems, and serial communications — the hardware-software interface of robotics.

## Resources

- **Arduino Language Reference & Documentation** — Arduino
  https://docs.arduino.cc/language-reference/
  The official reference for the Arduino programming language (functions, variables, structure) plus hardware guides — the most accessible entry point for microcontroller programming.

- **Learn Embedded Systems Firmware Basics** — Soham Banerjee, freeCodeCamp
  https://www.freecodecamp.org/news/learn-embedded-systems-firmware-basics-handbook-for-devs/
  A free handbook covering microcontroller memory layout, interrupts, timers, UART/serial, bare-metal vs RTOS, and clean embedded architecture.

- **Introduction to Embedded Systems Software and Development Environments** — Alex Fosdick, University of Colorado Boulder (Coursera, free to audit)
  https://www.coursera.org/learn/introduction-embedded-systems
  A beginner course on ARM microcontrollers, GNU toolchain build systems, Git, and memory manipulation.

- **Real-Time Embedded Systems Specialization** — Sam Siewert, University of Colorado Boulder (Coursera, free to audit)
  https://www.coursera.org/specializations/real-time-embedded-systems
  A four-course series on real-time scheduling theory (rate monotonic, dynamic priority) with hands-on Linux/RTOS and Raspberry Pi projects.

- **Embedded Systems Essentials with Arm** — Arm Education
  https://www.arm.com/resources/education/online-courses/efficient-embedded-systems
  A foundational course on Arm Cortex-M microcontrollers, RTOS, and serial communication.

## Notes

- Coursera courses are free to **audit** (full materials/videos), but graded assignments and certificates require payment or financial aid.
- Arm has announced Mbed OS and the Mbed Platform reach end-of-life in July 2026; treat the Arm course's Mbed-specific tooling as transitional and focus on its transferable Cortex-M concepts.
