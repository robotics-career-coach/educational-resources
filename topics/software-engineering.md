# Software Engineering for Robotics

Resources for Python, C++, Git/GitHub, and CI/CD — the core software skills needed to write, version, and ship robotics code.

## Resources

- **CS50's Introduction to Programming with Python (CS50P)** — Harvard University (David J. Malan)
  https://cs50.harvard.edu/python/
  A free, beginner-friendly OpenCourseWare course teaching Python from scratch including functions, exceptions, unit tests, and file I/O.

- **Learn C++** — learncpp.com
  https://www.learncpp.com/
  A free, frequently-updated, comprehensive modern-C++ tutorial site that takes total beginners through writing, compiling, and debugging C++ — essential since ROS 2 nodes are commonly written in C++.

- **Pro Git (2nd edition)** — Scott Chacon & Ben Straub (Apress, CC-licensed)
  https://git-scm.com/book/en/v2
  The complete, free, official Git book covering commits, branches, merging, and GitHub workflows.

- **CS50's Web Programming — Week 1: Git** — Harvard University
  https://cs50.harvard.edu/web/weeks/1/
  A free standalone lecture on Git/GitHub fundamentals (commits, branches, forks, pull requests) with video and slides.

- **GitHub Skills** — GitHub
  https://skills.github.com/
  Free interactive courses built by GitHub for learning Git, GitHub, and GitHub Actions hands-on inside real repositories.
